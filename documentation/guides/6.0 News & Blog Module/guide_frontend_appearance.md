# Guide to Page Appearance & Structural Layouts

This document maintains visual ASCII blueprints for the various page templates defined in the CSS Architecture. These diagrams dictate the HTML structural constraints (grid flow), ensuring consistent visual identity across the public-facing site. It is the source of truth for the appearance of the public facing pages.

---

## 6.0 News & Blog Module
**Scope:** Combined News & Blog Landing Page (6.1), News Feed (6.2), Blog Feed (6.3), Individual Blog Post View (6.4).

### 6.1 Combined News & Blog Landing Page
**HTML:** frontend/pages/news_and_blog.html
**CSS:** css/6.0_news_blog/frontend/news_blog_landing.css
**JS:** js/6.0_news_blog/frontend/news_snippet_display.js, js/6.0_news_blog/frontend/blog_snippet_display.js

```text
+--------------------------------------------------------------------+
| [Invisible Header: SEO, robots, agent metadata]                    |
|--------------------------------------------------------------------|
| SITE LOGO | [ Search Bar ] | Navigation Links                       |
|--------------------------------------------------------------------|
|           |                                                         |
| [Sidebar] |  NEWS & BLOG                                           |
|           |                                                         |
|           |  +-- Latest News ------------+  +-- Latest Blogposts--+ |
|           |  | +------+----------------+ |  | +------+----------+ | |
|           |  | | thumb| Title      Date| |  | | thumb| Title    | | |
|           |  | |      | Snippet text   | |  | |      | Date     | | |
|           |  | +------+----------------+ |  | +------+----------+ | |
|           |  | +------+----------------+ |  | +------+----------+ | |
|           |  | | thumb| Title      Date| |  | | thumb| Title    | | |
|           |  | |      | Snippet text   | |  | |      | Date     | | |
|           |  | +------+----------------+ |  | +------+----------+ | |
|           |  |  (... up to 5)           |  |  (... up to 5)      | |
|           |  |                           |  |                      | |
|           |  | [View all news ->]       |  | [View all posts ->]  | |
|           |  +---------------------------+  +---------------------+ |
|           |                                                         |
|--------------------------------------------------------------------|
| [Universal Footer]                                                  |
+--------------------------------------------------------------------+
```

**DOM Structure (generated by JS):**
- Two-column CSS Grid (.news-blog-landing): columns stack vertically under 800px
- Each column a <section class="news-blog-landing__column">
- Column heading <h2 class="news-blog-landing__heading"> wraps <a> link to full feed
- Snippets rendered as <div class="news-blog-landing__snippet"> with children:
  - Optional <div class="news-blog-landing__thumb-wrap"> containing <img class="news-blog-landing__thumbnail">
  - <div class="news-blog-landing__snippet-body"> containing:
    - <p class="news-blog-landing__date">
    - <h3 class="news-blog-landing__title"> with optional <a class="news-blog-landing__link">
    - <p class="news-blog-landing__text">
- "View all" link: <a class="news-blog-landing__view-all">
