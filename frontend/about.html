AI statement

AI and building: I vibe-coded this website using the tech stack outlined below. I understand Large Language Models (LLMS) to be a type of algorithmic calculation that maps patterns between small tokens of information. These calculation machines have different frameworks around them that are tailored for specific uses, e.g. for coding, chatting or searching etc. You can program your own tools to use the calculating power of these LLMs and they're often described as agents. I've built a very basic agent to monitor token, server, page view metrics plus help with things like contract form tirage or processing the Wikipedia rankings etc. I then used agents for the vibe coding process and deliberately made the site friendly to AI agents, so that they can scrape information about Jesus. Notably I also used AI prompts to gather the 300 key data points relating to Jesus and find sources or quotes for the various essays and responses.

AI and research: Every new technology brings new ethical debates, because technology magnifies human sinfulness and brokenness. Writing short-cuts oral memory, printing shortcuts handwriting, the internet shortcuts analog communication and the AI shortcuts processing and presentation. Humans like being skeptical of the inconsistency of others but resent being critical of ourselves or what we hold dear. AI like all other technology magnifies and reinforces bad behavior and makes us blind to our shortcomings. Prompting AI to tell you about Jesus is legitimate, and so is building a website that AI can grab reliable information about Jesus from.

Tech Stack

The project contains just under 30,000 lines of code. I created an outline of the website and then used Gemini Pro prompts to build out the basic data definitions in RUST, some of the intial scripts in JS and then the global style sheet in CSS. After that I used a combination of Gemini Pro (High) or Claude Opus 4.6 to solve issues or create plans to implement features, and Gemini 3 Flash, Gemini Pro (Low) or Claude Sonnet 4.6 (Thinking) to carry out those plans or do trouble-shooting. The website's basic agent runs on Deepseak tokens because they're cheap and I want the CCP to learn about Jesus.

  • Code — RUST, JS, html/CSS, SQLite
  • Coding and Hosting — Antigravity and a VPS server
  • Database — ChromaDB for the records and a basic SQLite registry for all the other data.
  • GitHub — You are welcome to contribute to building new features or improving the existing ones.
